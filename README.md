# CustomMenu

**CustomMenu** — это удобный плагин для серверов CS2, который предоставляет игрокам быстрый доступ к важной информации: правилам сервера, контактам администрации и полезным ссылкам через интерактивное меню.

## Команды
| Команда | Описание | Доступ |
|--------|----------|--------|
| `!openmenu` | Открывает главное меню плагина | Все игроки |
| `mm_custommenu_reload` | Перезагружает конфигурацию плагина | Консоль / Администратор |

## Установка
1. **Скачайте зависимости**  
   [Utils](https://github.com/Pisex/cs2-menus)

2. **Скачайте последний релиз плагина**  
   [CustomMenu v1.0.0](https://github.com/ownerok/CustomMenu-cs2-plugin/releases/tag/v1.0.0)

3. **Распакуйте архив**  
   Скопируйте содержимое архива в папку `addons` вашего сервера.  

4. **Настройте сервер**  
После установки перезапустите сервер или выполните команду `mm_custommenu_reload`.

## Настройка

### Основной конфиг
Файл: `addons/configs/CustomMenu/config.ini`

```ini
"Config"
{
 "start_menu"	"welcome"		// Меню, которое открывается при входе игрока
 "show_death_message" "1"        // Показывать сообщение при смерти (1 — да, 0 — нет)
 "show_roundstart_message" "1"    // Показывать сообщение в начале раунда
 
 "Menus" // Список всех доступных меню
 {
 	"welcome"	""
 	"contacts"	""
 	"links"		""
 	"rules"		""
 }
}
